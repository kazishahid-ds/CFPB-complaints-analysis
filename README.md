# CFPB Consumer Complaints Analysis

## Overview
Stratified sample (300k from 13M) → comprehensive EDA → (... NLP etc. ...)

## Files
- `1 Main_dataset_to_sample.ipynb`: Main dataset's 13M records → 300k stratified sample (proportional Product×YearQtr×Region sampling)
- `2 EDA.ipynb`: Univariate, bivariate, narratives EDA of the 300k-sample dataset

## Key Findings
- Credit reporting: 62% → 90% recently
- 96%+ timely responses
- Narratives declining post-2022
- FCRA/legal keywords dominate narratives
